<section style="margin-bottom:20px;">
<h3 style="background:#000;color:#fff;padding:6px 10px;display:inline-block;font-size:0.95rem;">
1. Abstract
</h3>

<p>
The Sovereign Throw Node is a high-output portable loudspeaker system designed for controlled long-throw sound projection using analog signal paths, high-efficiency Class-D amplification, and rigid enclosure architecture.
</p>

<p><strong>Design priorities:</strong></p>
<ul>
<li>Transient accuracy under load</li>
<li>Controlled directivity</li>
<li>High SPL efficiency</li>
<li>Mechanical enclosure stability</li>
</ul>

<p>
The system is intended for field audio, small venue reinforcement, monitoring, and portable performance applications.
</p>
</section>

<section style="margin-bottom:20px;">
<h3 style="background:#000;color:#fff;padding:6px 10px;display:inline-block;font-size:0.95rem;">
2. System Architecture
</h3>

<p><strong>Signal Input Stage</strong></p>
<ul>
<li>Balanced line-level input (XLR/TRS)</li>
<li>High impedance buffering</li>
<li>Optional analog gain staging</li>
</ul>

<p><strong>Signal Conditioning Stage</strong></p>
<ul>
<li>Optional analog filtering network</li>
<li>HF attenuation (~16 kHz region)</li>
<li>LF shaping (~40–60 Hz region)</li>
<li>No mandatory DSP in baseline configuration</li>
</ul>

<p><strong>Power Amplification Stage</strong></p>
<ul>
<li>Class-D topology (TPA3255 class reference)</li>
<li>36V nominal rail operation</li>
<li>&gt;90% efficiency target</li>
</ul>

<p><strong>Acoustic Output Stage</strong></p>
<ul>
<li>Direct radiator or horn-loaded variants</li>
<li>Controlled dispersion (typically 60° × 40°)</li>
</ul>
</section>

<section style="margin-bottom:20px;">
<h3 style="background:#000;color:#fff;padding:6px 10px;display:inline-block;font-size:0.95rem;">
3. Power Architecture
</h3>

<ul>
<li>36V nominal lithium-ion system</li>
<li>10S2P reference configuration</li>
<li>30A continuous BMS protection</li>
<li>Rail decoupling capacitors for transient stability</li>
</ul>

<p>
System optimized for high peak current delivery with controlled thermal rise under sustained output.
</p>
</section>

<section style="margin-bottom:20px;">
<h3 style="background:#000;color:#fff;padding:6px 10px;display:inline-block;font-size:0.95rem;">
4. Enclosure Design
</h3>

<ul>
<li>15mm Baltic birch plywood enclosure</li>
<li>Internal cross-bracing to reduce resonance</li>
<li>Airtight cabinet construction</li>
<li>Acoustic damping material for internal reflection control</li>
</ul>

<p>
The enclosure functions as both structural housing and acoustic control volume.
</p>
</section>

<section style="margin-bottom:20px;">
<h3 style="background:#000;color:#fff;padding:6px 10px;display:inline-block;font-size:0.95rem;">
5. Performance Characteristics (Target Model)
</h3>

<ul>
<li>Frequency response: ~50 Hz – 18 kHz (driver dependent)</li>
<li>High SPL capability relative to size</li>
<li>Low distortion within rated power envelope</li>
<li>Controlled dispersion for medium-range coverage</li>
</ul>
</section>

<section style="margin-bottom:20px;">
<h3 style="background:#000;color:#fff;padding:6px 10px;display:inline-block;font-size:0.95rem;">
6. Design Principles
</h3>

<ul>
<li><strong>Efficiency over complexity:</strong> minimized processing stages</li>
<li><strong>Mechanical stability:</strong> enclosure rigidity as acoustic determinant</li>
<li><strong>Power headroom:</strong> prevents transient compression</li>
<li><strong>Reproducibility:</strong> standardized component behavior across builds</li>
</ul>
</section>

<section style="margin-bottom:20px;">
<h3 style="background:#000;color:#fff;padding:6px 10px;display:inline-block;font-size:0.95rem;">
7. Limitations
</h3>

<ul>
<li>Not optimized for wide stereo imaging</li>
<li>Performance varies by driver selection and tuning</li>
<li>Requires precise assembly for full performance</li>
<li>Thermal behavior dependent on load and ventilation</li>
</ul>
</section>

<section style="margin-bottom:20px;">
<h3 style="background:#000;color:#fff;padding:6px 10px;display:inline-block;font-size:0.95rem;">
8. Applications
</h3>

<ul>
<li>Portable live sound reinforcement</li>
<li>Small venue PA systems</li>
<li>Studio monitoring (near to mid-field)</li>
<li>Field playback systems</li>
</ul>
</section>

<section style="margin-bottom:20px;">
<h3 style="background:#000;color:#fff;padding:6px 10px;display:inline-block;font-size:0.95rem;">
9. Conclusion
</h3>

<p>
The Sovereign Throw Node is a modular high-efficiency audio system designed for reproducible acoustic output using standardized hardware components and analog signal integrity.
</p>

<p>
It prioritizes physical system behavior over software-based correction models.
</p>
